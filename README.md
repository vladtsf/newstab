### Requirements
* Node.js
* zip (for releases)

### Development
* clone this repo
* run `npm install && npm install -g brunch coffee-script`
* `brunch build` or `brunch watch`
* for release run `cake release`
* don't forget to update version in app/assets/manifest.json
